数据库迁移说明

  1. 打开 Visual Studio 工具 -> Nuget 包管理器 -> 程序包管理器控制台
  2. 运行命令 Enable-Migrations
  3. 运行命令 Add-Migration {Summary} 其中 {Summary} 为本次更新说明，必须是英文
  4. 运行命令 Update-Database 将本次更新应用到数据库

[表] 平台账户表(Platform_Account)

包含平台账号信息。

字段 类型 默认值 说明
AccountId long 主键
Username varchar(20) 用户名
Password varchar(50) 密码（可用于登录及其它操作验证）
Name nvarchar(20) 姓名
Gender bit 性别
Mobile varchar(11) 手机号
Email varchar(50) 邮箱
AvatarId GUID 头像图片ID
AvatarUrl varchar(100) 头像图片URL
Status short 状态（枚举：PlatformAccountStatus）
Remark nvarchar(500) 备注
CreatedTime datetime now() 创建时间
LastLoginTime datetime 最后登录时间
LastLoginIP varchar(63） 最后登录的IP地址(支持ipv4及ipv6)

[表] 银行资料(Bank)

包含微信钱包，支付宝钱包，国内各大银行等，在总平台进行配置。

字段 类型 默认值 说明
BankId long 主键
Name nvarchar(20) 银行名称（如：中国工商银行)
Phone varchar(20) 服务电话（如：95551）
IconId Guid 银行图标ID
IconUrl varchar(100) 银行图标URL
SortOrder int 0 排序值
CreatedTime datetime now() 创建时间

[表] 奖励规则表(BonusRule)

扫码支付奖励规则表，一个业务类型只允许存在一个规则为启用状态。

字段 类型 默认值 说明

Name varchar(50) 规则名称（程序使用）

Content nvarchar(MAX) 规则内容(JSON数据)
CreatedTime datetime now() 创建时间
UpdatedTime datetime 最后的修改时间
